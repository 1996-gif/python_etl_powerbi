## âš™ï¸ How the ETL works (simple)
1. **Extract**: script reads `data/raw/sales_data.csv`.
2. **Transform**: clean dates, fill/convert types, calculate `total_amount`, add `month` column.
3. **Load**: writes cleaned CSV to `data/processed/sales_cleaned.csv`.

Run locally with:
```bash
pip install -r requirements.txt
python scripts/etl.py
```

After running, use the `data/processed/sales_cleaned.csv` file as a data source in Power BI Desktop to build visualizations (sales by month, sales by product, top customers, region comparison).
